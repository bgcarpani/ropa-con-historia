Start the backend and frontend dev servers for the Ropa con Historia app.

1. Start the NestJS backend in the background:
   - Working directory: `backend/`
   - Command: `npm run start:dev`
   - Runs on port 3001

2. Start the Vite frontend dev server in the background:
   - Working directory: `frontend/`
   - Command: `npm run dev`
   - Runs on port 5173

After both are running, report the URLs:
- Backend API: http://localhost:3001
- Frontend: http://localhost:5173

If either server fails to start, show the error output.
